# Automação com PyAutoGUI - Pesquisa de Clima

## Descrição
Script em Python usando PyAutoGUI para automatizar pesquisa de clima no Google.

## Funcionalidades
- Abre navegador automaticamente
- Acessa Google.com
- Pesquisa "Clima hoje na minha cidade"
- Tira screenshot da previsão
